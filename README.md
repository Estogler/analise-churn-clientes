# 📊 Análise de Churn de Clientes

## 🎯 Objetivo
Analisar os fatores que levam clientes a cancelarem um serviço (churn) e prever esse comportamento usando Machine Learning.

---

## 🛠️ Ferramentas Utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset
Dataset de churn de clientes de telecomunicações.

---

## 🧹 Tratamento de Dados
- Conversão de dados para formato numérico
- Remoção de valores nulos
- Transformação de variáveis categóricas
- Remoção de colunas irrelevantes

---

## 📊 Análise Exploratória
Foram criados gráficos para entender padrões de churn:

- Churn geral
- Churn por gênero
- Tempo de permanência vs churn
- Tipo de contrato vs churn

---

## 🤖 Machine Learning

### Modelos utilizados:
- Regressão Logística
- Random Forest

### Resultados:
- Acurácia: ~79%
- Melhor desempenho na identificação de clientes que não cancelam
- Dificuldade maior em prever clientes que cancelam

---

## 📈 Principais Insights

- Clientes com contratos mensais têm maior taxa de churn
- Clientes com menor tempo de permanência cancelam mais
- Clientes com maior cobrança mensal tendem a sair
- O modelo consegue prever churn com boa precisão, mas ainda pode ser melhorado

---

## 🚀 Conclusão

O projeto mostrou que é possível identificar padrões de cancelamento de clientes e prever esse comportamento com Machine Learning, ajudando empresas a tomar decisões estratégicas.
